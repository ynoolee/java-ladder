## 연습 : 자비지기_LadderGame

자바지기님의 영상 에서의 사다리게임 초기 구현을 생각할 때 , 생각해 볼 수 있는 케이스들

- n 명/높이 1/선이 없는 경우
  - 라인 n 에 대한 결과 는 n 이다

- 2명/높이1/선이1개
  ```markdown
  1 1
  ```
  - 결과
    - 0 -> 1
    - 1 -> 0

- 2명/높이2/선이1
  ```markdown
  1 1
  1 1
  ```
  - 결과
    - 0 -> 0
    - 1 -> 1
- 3명/높이1/선이1
  ```markdown
  1 1 0
  ```
  - 결과
    - 0 -> 1
    - 1 -> 0
    - 2 -> 2
- 3명/높이2/선이2
  ```markdown
  1 1 0
  0 1 1
  ```
  - 0 -> 2
  - 1 -> 0
  - 2 -> 1
