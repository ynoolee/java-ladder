
# Step 1 


## LadderGame

- n 명/높이 1/선이 없는 경우
  - 라인 n 에 대한 결과 는 n 이다

- 2명/높이1/선이1개
  ```markdown
  1 1
  ```
  - 결과
    - 0 -> 1
    - 1 -> 0

- 2명/높이2/선이1
  ```markdown
  1 1
  1 1
  ```
  - 결과
    - 0 -> 0
    - 1 -> 1
- 3명/높이1/선이1
  ```markdown
  1 1 0
  ```
  - 결과
    - 0 -> 1
    - 1 -> 0
    - 2 -> 2
- 3명/높이2/선이2
  ```markdown
  1 1 0
  0 1 1
  ```
  - 0 -> 2
  - 1 -> 0
  - 2 -> 1
